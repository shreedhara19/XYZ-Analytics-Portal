# XYZ Analytics Portal

XYZ Analytics Portal is a web-based analytics solution designed to visualize retail Sales and Quality performance data.
It integrates interactive Tableau dashboards within a React application to provide business users with actionable insights through a clean and intuitive interface.

---

## Design Choices

- Applied provided XYZ brand colours and logo across dashboards and UI
- Used multiple visualizations (KPI, bar chart, heatmap, treemap, line chart)
- Designed simple, business-friendly layouts for easy understanding
- Implemented sidebar navigation for smooth routing
